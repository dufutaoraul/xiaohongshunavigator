💡 可能的解决方案

🔧 根据诊断结果，可能需要以下操作：

  • 如果是网络连接问题：
    • ✅ 开启 VPN 或翻墙工具
    • ✅ 尝试更换网络环境（如手机热点）
    • ✅ 检查防火墙设置
  • 如果是 API 密钥问题：
    • ✅ 检查 .env.local 中的 DIFY_API_KEY
    • ✅ 确认密钥没有过期或被撤销
    • ✅ 联系 Dify 服务提供商
  • 如果是代理设置问题：
    • ✅ 配置系统代理设置
    • ✅ 在代码中添加代理配置
    • ✅ 检查企业网络限制

🔧 手动测试工具

您可以在浏览器开发者工具的控制台中运行以下命令进行手动测试：

// 1. 测试域名解析
fetch('https://pro.aifunbox.com/v1/workflows/run', { method: 'OPTIONS' })
  .then(() => console.log('✅ 域名可以解析'))
  .catch(() => console.log('❌ 域名解析失败，可能需要翻墙'))

// 2. 测试完整 API 调用
fetch('https://pro.aifunbox.com/v1/workflows/run', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer app-EZkRHFBBvn8qrIpHoXWdPnAG'
  },
  body: JSON.stringify({
    inputs: { test: "hello" },
    response_mode: "blocking",
    user: "test"
  })
}).then(r => r.json()).then(console.log)
